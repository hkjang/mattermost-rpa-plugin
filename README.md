# Mattermost RPA Analytics

조직 내 Mattermost 대화에서 IT 업무와 RPA 운영 관련 키워드, 주제, 요청 패턴을 분석해 운영 효율과 협업 품질을 높이는 플러그인입니다.

## 제공 기능

- 공개 채널, 비공개 채널, 스레드 메시지 분석
- 장애, 배포, 인프라, 네트워크, 데이터베이스, 보안, 개발, DevOps 등 카테고리 자동 분류
- 태그 클라우드, 키워드 빈도, 채널 비교, 핫토픽, 알림 패널
- 기간별 재색인과 JSON/CSV 리포트 추출
- 내장 사전 기반 분석, 제외 규칙, 보존 정책, vLLM 기반 AI 운영 요약

## 기본 분석 체계

플러그인은 아래 구조의 기본 사전을 포함합니다.

- 대분류: 장애, 배포, 인프라, 네트워크, 데이터베이스, 보안
- 확장 분류: 개발, CI/CD, 모니터링, 클라우드, 계정, 운영요청, 성능, 협업, 대응, 데이터, 보안운영, DevOps
- 각 항목은 `대분류 -> 중분류 -> 키워드 -> 활용 목적` 구조로 관리됩니다.

## 관리자 화면

- 분석 대상 팀/채널 설정
- 제외 채널, 봇/시스템 메시지 제외 규칙
- 알림 규칙 관리
- 태그 클라우드, 카테고리 분포, 채널별 통계, 핫토픽
- 키워드 상세 메시지 조회 및 리포트 추출
- 현재 필터 범위 기준 vLLM AI 요약 생성
